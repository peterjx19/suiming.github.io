《学习之道》(六)工具

发表于 2022-05-17 更新于 2022-05-30

小技巧

  1. 整理环境、冥想避免干扰
  2. 改造为负面思维为正面思维
  3. 有负面小情绪可理解，立刻开始做
  4. 不开电脑、先看简单的题、抄下要解决的题随身带着直到脱不下去了。

行动日记

  1. 简单列出下个星期要做的关键任务清单，一周一次。
  2. 在第一天写出第二天的日行计划清单
  3. 每天都专注今天的任务。
  4. 先从最困难的事做起

玛丽查的数学浸泡实践

  1. 学习材料的某小节，所有题目做一遍，每种类型都练到
  2. 浸泡难题
  3. 用表格列出所有重要概念，给各类问题配上例题
  4. 考前列一张内容全面的表格，包含科目、小结题型、解题技巧

克服拖延

  1. 记录行程日记
  2. 每天承诺完成一定任务
  3. 睡前写下计划的任务
  4. 将工作细化成一系列笑挑战
  5. 相信自己的新时间系统
