《学习之道》(二)专注模式和发散模式如何切换

发表于 2022-05-17 更新于 2022-05-30

两种模式切换帮助你掌握新知识

专注模式调动全部注意力思考特定类型问题，消耗意志力，然后用发散模式奖励自己。

发散模式能让学习更有深度和创造力。发散模式用新角度看，也会整合到已知的相关信息上

学习即创造： 放下手中事喘口气，发散模式就会乘虚而入，高屋建瓴搜寻解决方法。

激发发散模式的活动

健身、运动、慢跑、散步、游泳、跳舞、开车、涂鸦、沐浴、音乐、演奏、冥想、睡觉（终极法宝）

练习

棘手的问题，尽力做上几分钟，做不下去就做另一项任务，发散模式会自动执行

学习的小技巧

  1. 放下脚步，细嚼慢咽学的更深入
  2. 避开思维定式，做作业和考试，接受第一个想法有碍于另觅佳境； 闭眼：放松注意力，眨眼可短暂断开专注模式视角链接
  3. 解除难题，需要无意识思考的时间段
  4. 专注模式之间，要间隔几个小时，但是学习新概念不要隔天
  5. 愤怒和失意，不断的挫败感是暂停的信号，提示你要切换到发散模式
  6. 倾听：旁观者清，先把遇到的问题大卸八块，基本概念印刻于心，更容易接受别人的解释

工作记忆

大脑对正在处理的信息进行瞬时和有意识的加工，叫工作记忆（一次只扔4个球的杂耍员）

长期记忆

大量知识仓库，需定期回访保持新鲜

  1. 推导
  2. 定期重复有助工作记忆转移到长期记忆

睡眠

  1. 减缓呼吸、注意力放在呼吸上
  2. 戴眼罩或光线暗下来，保证安静环境。

睡眠好处

  1. 清除脑脊液毒素（阿兹海默和抑郁症）
  2. 加强记忆，巩固难点，加深知识回路
  3. 加快理解、拼凑答案，知识如梦，巩固记忆
  4. 1h充沛精力 > 3h疲劳大脑

小结

  1. 数学科学难题，专注模式打前阵，一轮战斗后切换到发散模式
  2. 挫败感是暂停的信号，应切换到发散模式
  3. 每天进步一点点。
  4. 让两种模式有足够的时间各行其事

《学习之道》(一)序 专注、发散思维

发表于 2022-05-17 更新于 2022-05-30

序

这是一本介绍方法、理论、案例、练习，可重复读的工具书

该书对如何吃透材料、集中注意力、抽空练习、提炼概念有详尽的描述
并认为每个人都有非凡的心算能力，自己没有数理天分的看法是完全错误的。

概念

专注思维

理性、连贯、分解的思维模式，由大脑前额叶皮质参与

发散思维

全局视角，创造力是在发散模式下激发的

为什么数学和科学很难对付

  1. 抽象和隐晦，需要额外练习，需要专注思维（基础）
  2. 解决问题时，思维定式下专注模式下离正确答案远。需要切换发散模式
  3. 如果基础不牢固就开始发散，会导致更加严重后果。

任何学科的问题都需要专注模式和发散模式之间相互切换，一个模式接受信息处理再传到另一个模式处理，大脑要解决问题离不开信息之间的来往传递，这对于理解数据和自然科学尤为重要，对语言、音乐、创意写作同样奏效

如何进入专注模式——番茄钟

  1. 关掉干扰源（手机、屏幕、声音）
  2. 25分钟全神贯注，不用在意是否能完成
  3. 25分钟后奖励自己刷手机
    练习：想一个最重要的事情，用一个番茄钟去完成它，每天重复三次

如何理解复杂困惑的概念

  1. 每章节读几遍，不按顺序。了解大概在说什么，然后再去读细节。
  2. 如还是不明白，就google搜索，换个思维和思路
  3. 发散思维，兜兜风，散散步

《如何阅读一本书》(四)主题阅读

发表于 2022-05-17 更新于 2022-05-30

主题阅读的两个要求：

对特定的问题，所牵涉的不是一本书

就总的来说，应该读那些书。

主题阅读中检视阅读的必要性：

  1. 在读之前并不清楚主题，在你开始读了之后，你可能才明确你的主题是什么？
  2. 检视阅读是进入主题阅读的前置作业和准备作业。收集好书目后，做检视阅读，需要做到如下两点：
  3. 对研究的主题有清晰的概念
  4. 简化书目到合理程度

主题阅读的五个步骤：

一、找到相关的章节

  1. 在过滤了相关的书后，把这些书检视阅读一遍，找出书中与你主题和问题相关的章节

  2. 不建议找相关的书、主题相关的章节这两个步骤一起做

打个比方 For循环嵌套会导致 O(N) * O(M), 而分两步的两个For循环复杂度是O(N) + O(M), 可大大减少心智压力。

二、 引领作者达成共识

分析阅读是直接使用作者的关键字，而主题阅读是要用你自己的关键字诠释作者的关键字。

在代码中就是多一层抽象，这层抽象是防腐层，设计模式中叫里式替换。

三、厘清问题

第二步是找关键字并抽象，本步骤是根据关键句找出一组不偏不倚的主旨，这些作者需要解决的问题。

四、 界定议题

对主题的争议，可能是赞成或反对，也可能有多个解答，我们要把前后关系整理清楚。

五、 分析讨论

前四个步骤解答的问题是：这些书在说什么，是如何说的？

我们还剩下两个问题，这是真的吗？与我何干？

我们不仅仅要问问题和回答问题，我们要依照特定顺序提问题，并知道为什么是这个顺序。

能说出这些问题的不同答案，并说明原因。我们可以依据此做出研究的基础，让后人少走弯路。这就是主题阅读的价值

《如何阅读一本书》(三)分析阅读的三阶段和阅读目标

发表于 2022-05-17 更新于 2022-05-30

分析阅读第一阶段

规则一：这本书的种类和主题是什么？ 是实用性还是理论性

规则二：使用单一的句子，或最多几句话叙述整本书的内容

规则三：将书中重要篇章列举出来，说明他们如何按照顺序组成一个整体架构。
写作是有纲要后，在架构上铺设血肉，尽量把纲要隐藏起来，给全书增添必要的空间和深度。

规则四：找出作者要问的问题

分析阅读第二阶段

规则五：找到关键字，通过他们于作者达成共识

如几何原理的「点」「线」「面」，财富论里的劳工、资本、土地、薪资、利润。需要明确地、逐字逐句的理解清楚。

规则六：从最重要的句子中抓出作者的主旨

  1. 作者标注的句子；
  2. 你读起来很困惑的句子；
  3. 在仔细分析后，要用自己的话让别人也明白。

规则七：找出作者的论述，重新架构这些论述的前应后果

一个或多个特殊事实证明共通的概念是归纳法；

连贯的通则证明进一步的概念是演绎法

规则八：确认作者已经解决了什么问题？还有哪些是未解决的？

哪些是作者认为无法解决的

分析阅读的第三个阶段：像沟通知识一样评论一本书

九：除非完成了大纲架构阅读，并能读懂了整本书，否则不要轻易评论。

批评之前，应该对整本书都了解透彻。

十：不要陷入情绪和偏见，虽然这无可避免，不要争强好胜。

我们的目标是增进知识，而不是取得胜利。

十一：说出评论时，要证明自己能够区别出知识和个人观点的不同。

知识是能够被证实大家认可的，观点是每个人的想法。

批评作者观点的要求：

十二：证明作者知识的不足。

针对作者提出问题的解法中，作者缺失了哪块的知识，与解决方法强相关。

十三：证明作者知识的错误

在作者论述中的某一条，本身是错误的，导致整个推导是错误的。

十四：证明知识的不合逻辑

作者未合理的使用逻辑关系：如必要条件变成充分条件、整体和局部的不合理替换。

十五：证明作者分析和理由是不完整的。

作者的分析和理由，和作者想要解决的问题不相关。

辅助工具：

一、导读和摘要：

导读和摘要可能是错误的，不完整的，所以我们要在自己完成阅读后再去看导读和摘要

正确的做法是：在我们读过这本书，用导读和摘要唤起我们的回忆；在主题阅读时，查看是否和主题相关。

二、字典：文字是物质的；文字是语言的一部分；文字是符号的；文字是约定俗成的。

三、百科：事实是一种说法；一种证实的说法；真相的反映；某种程度上的约定俗成。

阅读的目标：

许多作品是有关联的，他们是基础、中级、高阶的关系。

同时也有许多作品是有时间顺序的，他们之前是连贯的。如哲学作者，即是作者也是读者。

通过努力地阅读，读者达到和作者同一个水平，他们应该能够平等对话。

如何说一个人读书读的好，应该偏重质而非量，好书可以一遍一遍地读，而有些书只需要随便翻翻就可以了。

温故而知新，可以为师矣，在某一领域反复读和揣摩，你自然可以变成这一领域的资深乃至专家。

《如何阅读一本书》(二)阅读速度原则、怎么消化一本书、阅读误区

发表于 2022-05-17 更新于 2022-05-30

正常情况，一个人文素养优良的高中，应该能培养出能达到分析阅读的读者
一个优秀大学，也该培育出能进行主题阅读的读者，但实际上现在很多人的阅读水平普遍偏差。

用什么阅读速度去阅读?

  1. 阅读速度的原则是：慢不该慢到不值得，快不该快到有损满足和理解。
    阅读是在限定时间内完成知识的吸收，核心就是时间和吸收，在保证理解的情况下越快越好。
  2. 加快阅读速度的方法有： 用手指指着内容移动
  3. 一个误区是越快越好，不管有没有理解。
    对于经典可以反反复复慢慢理解，对于快餐知识可以快速浏览，不同材料

检视阅读怎么做？

一、有系统的略读和粗读

  1. 书名、目录页 、索引、封皮。
  2. 挑主题息息相关的篇章、东翻翻西翻翻。
  3. 尝试找主题、思想和作者描述重点。

检视阅读的速度为什么快？

并不是阅读速度快，而是读的内容少：在一定时间内跳着读，只读了书本的很少一部分，大致了解本书的重点。

粗浅的阅读怎么做？

非常重要的阅读规则：头一次面对一本难度的书的时候，从头到尾读一遍，遇到不懂的地方千万不要停下来查询和思索

直到读到看得懂的地方，集中精力看这部分。喜剧和论说性的书籍尤其如此。

做自我要求的读者需要解答四个问题

  1. 整体来讲，这本书谈什么？
  2. 作者细说了什么，怎么说的？找到想法、申明、论点
  3. 这本书有道理吗，是全部有道理还是部分有道理？
  4. 这本书跟我有什么关系？给我什么讯息，对我的意义是什么？

检视阅读要回答第一、二个问题，在分析阅读的时候要解答第三、四个问题。

怎么消化一本书？

检视阅读要做结构笔记，用户描述全书架构。

分析阅读要做概念笔记，把细节、论述过程、重点知识点描述清楚。

阅读并不仅仅用眼睛看，你要看完后想，想完之后用自己的话说一遍，说完之后还要在日常的工作生活中练习，至此才算是内化了这本书。

阅读技巧怎么练习？

检视阅读、分析阅读、主题阅读，初期按照步骤一步步地做.

当你熟悉了这套流程，把规则连贯起来。就像游泳和跳舞一样，动作分解后连贯，阅读的行为比游泳跳舞会复杂一些。

《如何阅读一本书》(一)做一个有要求的读者

发表于 2022-05-17 更新于 2022-05-30

阅读的目的是什么？

  1. 获得知识
  2. 获得心灵的和精神的成长

怎么样才是主动阅读？

需要按照一定的方法，去读出作者的意图

比如超越基础阅读，做检视阅读的时候需要出四个主要问题：

  1. 总体讲，这本书在谈些什么？

  2. 这本书细说了什么，他的想法、声明和论点是什么？

  3. 这本书讲的是否有道理？

  4. 这本书内容和我有什么关系。

作者为什么认为这件事很重要，如果我收到了启发，我找出了其他更深沉的含义和建议吗，以获得更多的启发？

如何让一本书真正属于你？

你必须读出言外之意，才能获得更大的收货。

我们要保持清醒，并主动地思考，把感想写出来，概括作者的思想。

写下来时再表达你与作者之间相同或不同的观点。

就像和作者对话一样。

怎么做标记？

单底线（重点，有力量的句子）

双底线（强调）

空白处☆ （最重要的十个段落）

空白处编号（论点的陈诉）

空白处页码CF（相同论点）

圆圈（重点）

空白处笔记（阅读的问题或答案，概括论点）

最后一页（个人索引页）。

三种做笔记方法

  1. 检视阅读：在目录页、书名页按这个结构编写（这是什么书；整本书谈什么；用什么整体架构发表观点)

  2. 分析阅读：概括作者的观点，写出你的观点

  3. 主题笔记：把所有关于这个主题的相关陈述和疑问顺序排列

怎么培养阅读的习惯

先形成规则，把多个单个动作组合成多个动作。多个动作连贯起来融为一体，再专注到阅读目标上。

自动反应

很多时候，我们在对某人或者某事做判断的时候，并没有用上所有可用的相关信息。

相反我们只用到了所有信息里面最具代表性的一条

尽管只靠鼓励数据容易做出愚蠢的决定，可现代生活的节奏有要求我们频繁使用这一捷径。

靠着成熟而精密的大脑，我们建立了一个信息繁多的快节奏复杂世界，是的我们不得不越发依赖类似动物的原始应对方式来对应它。

捷径应受到尊重:

如果顺从业者公平公正地利用我们的捷径响应方式，我们就不应该把他们看成敌人。

事实上他们是盟友，我们能更方便地开展高效率、高适应度的生意往来，只有通过弄虚作假、伪造和歪曲证据误导我们

还击恶意使用快捷响应的人

我们采取一切合理的方式，抵制、对峙、谴责、抗议来报复以刺激我的捷径反应作为目的的虚假信号

《影响力》第七章-稀缺

发表于 2022-05-06 更新于 2022-05-30

机会越少，价值似乎就越高：

对于失去某些东西的恐惧，似乎要比获得同一物品的渴望，更能激发人们的行动力。

假设瑕疵把某样东西变得稀缺了，垃圾也能变成之前的宝贝。（收藏家珍贵的错误）

对于稀缺的应用：

一个夫妻对一件商品感兴趣，销售员说这个商品已经卖给被人了，提升稀有程度。

同时话术很重要：“我可以去查查，但是我能不能认为你们真的很想要它，我找到了一台你们会买下它”

这个时候这个商品在夫妻的眼中就很奇妙了，很容易做出承诺。

接下来找到商品，依据承诺一致的原则，大概率成交了。

“专场放映，座位有限，预订从速，过时不候”

现在很优惠，以后没这么大力度了

多买，以后买不到了

现在买，以后我不来了。

逆反心理的核心：

我们依据获得一样东西的难易程度，迅速准确地判断它的质量

机会越少，我们的自由随之丧失。

保住既得利益的愿望，是心理逆反理论的核心。

无论是成年人还是青少年，都受逆反心理影响：

对于两岁的小朋友和青春期的青少年，逆反是最严重的。

越是干涉越容易起到相反效果。

尽管家长干涉令感情关系出现某些问题，比如更多挑剔、挺多报告负面信息，但干涉让情侣双方觉得彼此更加相爱，更想结婚了。

成年人也是这样，立法要求所有人持枪，反而降低了持枪率。

立法禁止酒精，反而增加了酒精的销量。

信息审查真的能实现它的预期目的吗？

每当东西获取起来比以前难，我们拥有它的自由受到了限制，我们越发地想要得到它。

信息审查对受众的影响，最耐人寻味的一点倒不是受众比以前更渴望信息了，而是人们对得不到的信息变得更加接受和包容。

信息限制让受禁的信息更有价值，想让信息变得宝贵，不一定非要封杀它，只要把它弄成稀缺就行了。

服从权威的现象

米拉格拉姆电击实验：

找志愿者当老师，请演员当学生，研究员命令老师：只要学生答错题目，就加大电击的强度。

结果表明100%的老师受试者都遵守了研究院的命令。即使学生苦苦哀嚎，即使为受害者遭遇感到痛苦。
