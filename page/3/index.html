《微习惯——简单到不可能失败的自我管理法则》

发表于 2022-05-29 更新于 2022-05-30

微习惯

看上去是一个自我管理系统，从小习惯开始

目录：

  1. 微习惯是什么：习惯和大脑；培养好习惯
  2. 大脑的工作原理： 稳定大脑；聪明管理者；自动行为放逾期；模式探测器
  3. 动力VS意志力： 意志力打败动力原因；意志力工作原理
  4. 微习惯策略：运用意志力；拓宽舒适区；出现阻力时间点；跨越障碍
  5. 微习惯独特之处：现有习惯；微习惯+意志力；没有截止时间；提升自我效能感；自主感
  6. 彻底改变的八步骤：选择微习惯和计划；挖掘内在价值；纳入日程；回报机制；跟踪；微量开始超额完成；服从计划；留意习惯标记
  7. 微习惯八大规则：不自欺；满意每个进步；回报自己；保持头脑清醒；
    感觉强烈抵触后退缩小目标；提醒自己很轻松；不要小看；多余精力超额完成任务

大脑组成和微习惯原理

大脑由执行决策和自我行为模式识别两部分组成。

  1. 前额皮层管理功能相当活跃，但是需要消耗大量精力

  2. 基底神经节的自动功能，效率高，无需持续监督能处理各种任务。

微习惯策略的重点

远离消极的东西；靠近积极的东西

给生活注入好习惯，被动坏习惯通常会被替换。

微习惯，小的不可思议的一小步，可以触发正反馈，提升自我效能感，减轻抵触情绪，获得惯性。

压力大会促使你使用习惯做法，如果习惯很好，对自己来说是非常好的。

大脑分工和微习惯关系

大脑前额皮层容易疲劳，功能强大，消耗太多精力，当疲劳时，掌管重复的部分会接管大脑。这块是基底神经节，一旦我们训练基底神经节完成积极的重复行为，我们就能获得巨大的收益。

微习惯和意志力

越缺乏动力，你消耗的意志力越高；意志力消耗太高，你就难以持续。

  1. 动力并不可靠： 精力不足；很难培养；你不想；热情递减。所以不要依赖动力，有很好，没有也没关系。
  2. 意志力很可靠：能自我强迫；可以强化；可以通过计划执行。

意志力损耗的5个因素：努力程度、感知难度、消极情绪、主观疲劳、血糖水平。

微习惯几乎不会消耗意志力，能保证你的意志无穷无尽，同时在任何情况下都能启动一个微习惯。

  1. 努力程度： 自我损耗极小
  2. 感知难度： 没有难度
  3. 消极情绪：你几乎感觉不到因为要做微习惯导致的消极情绪
  4. 主观疲劳： 微量疲劳，一想到微目标，就感觉精力充沛
  5. 血糖水平：在低血糖和疲劳是，最能让你动起来的是微习惯

微习惯和阻力

  1. 阻力： 启动阻力（初始速度） 继续行动阻力（加速度）
  2. 自主权是微习惯的另一个重要的核心。
  3. 微习惯带来自我能效感、自主权、远离恐惧胆怯和忧郁、增强正念

微习惯的八步走

  1. 选择三个微习惯，评估并做好长期计划
    把习惯变成小的不可思议，小到让你感到荒唐。
  2. 给习惯赋予意义
  3. 设定计划
  4. 确定回报
  5. 记录
  6. 微量开始，超额完成
  7. 服从安排，摆脱高期待
  8. 留意习惯养成的标记： 不抵触、认同自己、行动不需考虑、不担心、常态化
    即使没有超额完成，你也是了不起的，超额完成了，你是很了不起。

微习惯设定的重要原则

  1. 不自欺，不要偷偷提高标准
  2. 满意每一步，大脑错觉是微习惯的重要核心，是一种相信小积累能量变到质变
  3. 经常回报自己，恢复意志力
  4. 保持清醒，完成了任务兴奋，执行时单调和重复，都应该平常心
  5. 提醒自己很轻松
  6. 相信微步骤
  7. 不要制订更大的目标，而应该专注任务

《学习之道》(十)总结：十个好的学习法则、十个坏的学习法则

释放无尽潜力

10个好的学习法则

  1. 运用回想 （小学语文总结段落思想、中心思想）
  2. 自我测试
  3. 对问题组块
  4. 间隔重复动作
  5. 练习中交替不同的解法
  6. 注意休息
  7. 使用解释性的提问和简单的类比（费曼提出）
  8. 专注
  9. 困难的事情最先做
  10. 心理对照

10个坏的学习法则

  1. 被动重复阅读
  2. 满眼重点标记
  3. 瞟一眼解题思路，就胸有成竹
  4. 火烧眉毛才学习
  5. 清楚解法，仍反复练习
  6. 与朋友把学习变成闲聊
  7. 做题忽视课本
  8. 有疑问不找导师和同学核对并解决
  9. 时常分心，还以为自己学的深入
  10. 睡眠不足

小结

分类、感想

如果对书分类，这是一本工具书，又不局限于工具书，

有句话是「假传万卷书真传一句话」。

就是说如果仅仅是工具书，告诉你怎么做就好了。

但是我们都是凡人，没有那么高的悟性。作者要让你心服口服，就把原理和推导过程告诉你，让你自己得出结论说服自己，方能让你心服口服。

内容评论

这本书引入大量的论证和参考文献去解释每一条法则，最终就是这一页的内容。如果没有经过前面的学习，我会有很多疑问和踌躇：要么动作不标准；要么半信半疑地做。

价值

目前我吸取到对我最有帮助的习惯有：学习要专注理解、每小节总结、搭建组块、间隔地重复、简化组块模型。

反思我学习习惯，搭建组块和重复的事情很少做，所以学得多，记住的很少，效果很差。

在之后的学习中，要好好的用到这些原理，并经常翻翻这本书，内化这本书，养成良好的学习习惯。

认识自己大脑的运行方式，充分发挥大脑潜力。

《学习之道》(九)如何准备考试

发表于 2022-05-17 更新于 2022-05-30

备考检查清单(经常做才能回答是)

  1. 尽力理解课本内容（带目的找例题不算）。
  2. 和同学一起解决作业问题，核对答案。
  3. 和同学合作之前，尝试自己大致写出每道题的解法。

备考

回答越多是，说明备考越好，两个和两个以上否要改变思路

  1. 积极参与家庭作业小组讨论（贡献点子，提出问题）。
  2. 遇到困难请求老师和助教。
  3. 交作业时完全理解所有问题的答案。
  4. 不明白解法时，在课上提问需求解释。
  5. 学习指导手册，考前有仔细通读过一遍吗，确保手册上所有问题都会做了吗。
  6. 尝试快速列出解题纲要，而不在基础计算上花时间。
  7. 一起复习学习指导手册和问题，并相互提问。
  8. 考前有复习题，是否参加过，在课上提出任何你不太确定的问题。
  9. 考前有合理的睡眠。

更多技巧

  1. 有难入简，先做男难的，能做多少是多少；然后做简单的；然后再做另一道困难的。 保证发散模式能充分发挥作用。
  2. 考试前克服恐惧：把计划B写出来，写下临考前自己的感觉和想法。
  3. 考试中克服焦虑：把注意力转移到呼吸上，手放到腹部，体会手的起伏。
  4. 考前浏览复习材料，重温一遍。
  5. 考中从后往前检查，正面的念头。
  6. 做题时停顿几秒，如闭眼5秒钟。

《学习之道》(八)塑造你的大脑、学习的复兴

发表于 2022-05-17 更新于 2022-05-30

形成直观的认知

  1. 在熟练运用组块资料之后，尽量不要纠结于每一个小细节，而是自然而然地去解决问题。
  2. 有些同学能快速掌握材料，给别人打击，但是平均水平的学生在主动性上、做好事情能力上和创造力占优。
  3. 把握创造力的关键是专注模式切换到更放松、白日梦一样的发散模式。

事实

  1. 大脑发育速度不一样，有些人25岁才完成发育。
  2. 在科学界很多重量人物，起初都是问题少年。
  3. 在科学数学技术领域取得成功的专业人士，逐渐习得一个特质，就是如何组块——提炼关键思想。
  4. 比喻和实体类比也能构建组块，组块可使截然不同的领域概念对另一个领域产生影响。
  5. 无论做什么职业，都要有开放的心态，保证自己学习宝库常备数学和科学知识，存储更多组块。

抽象和简化概念

  1. 心灵之眼非常重要，帮我们在大脑中排演，将学过的知识拟人化。
  2. 迁移是把所学知识从一个知识背景运用到别处的能力。
  3. 关键是要掌握一个数学概念的精髓，有利于概念迁移和将之应用到新途径中。
  4. 学习过程中一心多用让学习无法深入，限制迁移所学知识的能力。

学习的复兴

  1. 自主学习是一种最深入、最有效的学习方式
  2. 提高独立思考能力
  3. 在学习中毅力往往比智力更加重要
  4. 锻炼自己去解除你敬仰的人，你会结识知识渊博的导师，一席话可能改变你的未来。
  5. 如果你没有快速掌握学习材料的重点，不要灰心，学的慢的同学会抓住根本问题，真正理解前因后果能更深层次地理解问题。
  6. 人人是竞争和合作的关系，总有人批评和低估你付出的努力，请淡然处之。

团队合作

  1. 即使自信一切没问题，专注模式可能在不经意犯下致命错误。
  2. 和愿意提出不同见解的小伙伴一起讨论，找出想法中的方法；随机应变应对压力；确保自己真的理解了你想让别人结识的知识，巩固已有知识；建立重要的职业关系
  3. 客观对待学习中的批评，不要自欺欺人

物理学教授布拉德*罗斯对学习的建议

  1. 检查计算过程花的时间比解决过程花的时间少很多，多检查。
  2. 量纲分析是你的朋友，错误极有可能发生在计量单位不匹配的关键点上。
  3. 思考方程的意义，对计算结果有预期。
  4. 复杂因子，变量分别取极限，看是否对你的理解方程有意义。

想起高中时候考试，我都是先把所有题目都看一遍，尤其是大题目，尝试写下是哪个模块下的什么思路。然后才从头做简单的题目。

记忆宫殿法

有助于记忆的形象嵌入熟悉的场景，让自己感受视觉记忆系统赋予的力量

理解后再记忆，对学习材料记忆犹新

空间思维的提升

  1. 童年时期打篮球、拆装玩具
  2. 准确地素描，玩3D电脑游戏
  3. 玩3D拼图
  4. 弃用DPS，用地图导航

练习

  1. 使用记忆宫殿
  2. 给同学讲述记忆宫殿法。

口述记忆宫殿法

  1. 挑选你熟悉的空间，如办公室布局，或者家庭布局，把它当成你的记事本
  2. 将要记忆的内容按顺序分别放到布局中的某个地方，并想象成一个鲜活的场景。
  3. 按照在空间布局中行进路径，多走几遍。

技巧

  1. 打造生动形象的比喻
  2. 间隔重复有助将概念向记忆存储
  3. 创建意群（如藏头诗一样，把整个内容抽成一句话）
  4. 别把窍门和真正知识混为一谈
  5. 讲故事
  6. 动手写写加强记忆
  7. 自言自语说给自己听
  8. 体育锻炼对神经元生长有很大促进作用
  9. 入睡前，回顾当天所学。

练习

拿出一张纸，给要理解的概念创建一个形象的类比和口诀

《学习之道》(四)组块的构建

发表于 2022-05-17 更新于 2022-05-30

解题有两种模式

按顺序推理和跳跃式的，而直觉是跳跃式的，大部分难题是由直觉解决的，需要用专注模式对直接进行验证，直觉并不总是正确的。

构建组块的重要性

  1. 一个过程或者思路理解透彻后，在宏观视角可视为一个整体，高度压缩后存进大脑，浓缩的思维让人获取洞察力
  2. 重复和练习的背后是固化组块，难点是枯燥
  3. 如果想对学习材料了如指掌，那么从上至下的宏观视角和从下至上的组块能力都不可或缺。

我们都喜欢创新，以及能够纵览全局的学习。但在数学和科学的学习中，必须进行适量的练习和重复，否则就无法构建组块来支撑专业技能科学的学习中，必须进行适量的练习和重复，否则就无法构建组块来支撑专业技能

提取知识

提取知识和回想知识帮助我们增加了学习深度，帮组我们实现组块。

如果一开始就画概念图之间关系，就是基础组块还没建立，就试着构建联结，完全是空中阁楼

活动一下，对难以理解的关键概念特别有用

整理组块

战胜自己， 整理课堂笔记和阅读笔记

与其过度学习，不如穿插学习

不要在同一个概念和技巧上投入太多时间学习，一旦理解了这个概念，就应该去学其他概念，隔几天在回顾这个概念。

如何搭建强有力的组块

  1. 全程纸上解决一个重难点问题 （要有线程答案，但是提交答案前不看）
  2. 重做一次，注意关键步骤
  3. 休息一下，启动发散模式
  4. 睡眠
  5. 再来一次，再次关注关键步骤（这个叫刻意练习）
  6. 给自己添加新题
  7. 主动重复，回想题目关键步骤

小技巧

  1. 越努力回想学习材料，你的记忆就越深刻；
  2. 提取练习是最强效的学习方式之一。
  3. 遭遇瓶颈，自然现象，越过后理解会更加深刻。
  4. 怎么备考，问题和答案梳理整齐，方便浏览；手写答案；主动重复。
  5. 测试是强效的学习：避免发蒙（工作记忆被填满），组块可以给工作记忆留出更多空间。

灵感项目经验

  1. 挖掘他人身上正能量特质
  2. 系统理解写下每个步骤
  3. 重复练习、发散模式专注模式切换

《学习之道》(三)组块

发表于 2022-05-17 更新于 2022-05-30

专注的练习和重复是创造记忆痕迹的过程，任何专业技能的培养都是积跬步已成千里的过程

组块是什么？

所罗门的组块难题

所罗门记者过目不忘，每条记忆痕迹都非常鲜明，妨碍了他对记忆痕迹真核，创造概念的块。

组块是根据意义将信息碎片组成的集合，组成一个有意义、便于记忆的组块。

思维组块是绝大多数科学、文学、艺术知识的构成基础

要熟练掌握数学和科学，就要创造概念组块，将分散的碎片组合起来，让大脑高效运转。而不必纠缠微观基础信息，提纲挈领。

组块的基本步骤

接触新知识，总是有线程的解题方法，首次尝试理解的认知负担很重。如陌生路上要开GPS，大量细节能够帮你看清关键特征和基本原理

专注

注意力集中在需要组块的信息上，创造新的神经模型、和既有模型联结（自下而上搭建组块）

理解

打包前要理解，专注和发散的交替思考，理清头绪、把握概念—— 课后要赶快复习，合上书本再找问题测验新学到的解题方法，提高学习效率（自上而下纵然全局）

背景

跳出初始问题，更宽广的视角看问题。推敲练习，有助于能用在哪，不能用在哪里， 会成为多条痕迹路径的驿站（熟悉背景构建联结）

技巧

睡觉前用当天知识为今天画上句号
宏观图景： 条理清晰演讲、章节大纲、流程图、表格、思维导图

能力错觉和回想

不断阅读笔记和课本，产生错觉知识已经进入大脑。 小和尚念经。

用对了学习策略，求知心才不会落个竹篮打水一场空。

学习技巧

  1. 划重点、下划线会误导和效率低下，欺骗你大脑在工作。 可以用六百
  2. 在学习中回想，让大脑提取关键概念而不是反复阅读，蒋更加高效学习
  3. 不看参考答案直到你交了作业
  4. 24小时内巩固一次，写一遍笔记或背诵，几天、几月、几周复习一遍，以后就不会忘记了。
  5. 不断地运用概念，直到变成自己第二天性，像工具一样信手拈来。
  6. 从笔记挑一个概念，读一遍，然后拿开，看自己能会议多少内容。再看笔记试着回忆一下，可大幅增进认识
  7. 专注钻研的部分，一但首页概念存入大脑，第二个概念进入大脑就容易一些，第三个也不会太难，这是努力让过程更轻松
  8. 搭建组块过程是蓄念大脑过程，电脑识别特定问题、形式和类别，能自动快速对症下药，构建了组块模型。

《学习之道》(二)专注模式和发散模式如何切换

发表于 2022-05-17 更新于 2022-05-30

两种模式切换帮助你掌握新知识

专注模式调动全部注意力思考特定类型问题，消耗意志力，然后用发散模式奖励自己。

发散模式能让学习更有深度和创造力。发散模式用新角度看，也会整合到已知的相关信息上

学习即创造： 放下手中事喘口气，发散模式就会乘虚而入，高屋建瓴搜寻解决方法。

激发发散模式的活动

健身、运动、慢跑、散步、游泳、跳舞、开车、涂鸦、沐浴、音乐、演奏、冥想、睡觉（终极法宝）

练习

棘手的问题，尽力做上几分钟，做不下去就做另一项任务，发散模式会自动执行

学习的小技巧

  1. 放下脚步，细嚼慢咽学的更深入
  2. 避开思维定式，做作业和考试，接受第一个想法有碍于另觅佳境； 闭眼：放松注意力，眨眼可短暂断开专注模式视角链接
  3. 解除难题，需要无意识思考的时间段
  4. 专注模式之间，要间隔几个小时，但是学习新概念不要隔天
  5. 愤怒和失意，不断的挫败感是暂停的信号，提示你要切换到发散模式
  6. 倾听：旁观者清，先把遇到的问题大卸八块，基本概念印刻于心，更容易接受别人的解释

工作记忆

大脑对正在处理的信息进行瞬时和有意识的加工，叫工作记忆（一次只扔4个球的杂耍员）

长期记忆

大量知识仓库，需定期回访保持新鲜

  1. 推导
  2. 定期重复有助工作记忆转移到长期记忆

睡眠

  1. 减缓呼吸、注意力放在呼吸上
  2. 戴眼罩或光线暗下来，保证安静环境。

睡眠好处

  1. 清除脑脊液毒素（阿兹海默和抑郁症）
  2. 加强记忆，巩固难点，加深知识回路
  3. 加快理解、拼凑答案，知识如梦，巩固记忆
  4. 1h充沛精力 > 3h疲劳大脑

小结

  1. 数学科学难题，专注模式打前阵，一轮战斗后切换到发散模式
  2. 挫败感是暂停的信号，应切换到发散模式
  3. 每天进步一点点。
  4. 让两种模式有足够的时间各行其事
