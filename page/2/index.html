《原则》作者经历(二)

发表于 2022-07-18 更新于 2022-09-29

2. 跨越门槛

进入大学

1967年到1979年，不良经济冲击，资产价格缩水，而作者不断买进，直到他明白了出 了什么问题： 股市反映了人们的预期。
秋天进入长岛大学就读，高中成绩C，但大学是自由的，可以学习感兴趣的东西，作者开始取得好分数。
学习冥想让作者更清晰、更开创性的思考，拥有平静的开放思维
这个期间是自由恋爱、毒品、拒斥传统权威的时代，他敬仰的乔布斯一样，不愿被人牵着鼻子，学习冥想、发挥想象力、倒灶新奇的好东西。
当时发生战争，并出现很多伤亡。他父亲帮作者免于服兵役，作者为无法给国家尽力感到愧疚，同时为自己未受到伤害感到宽慰
1971年被哈弗商学院录取，并获得纽交所大厅交易员的差事，欧洲人不愿意接受美国的美元了。
在哈弗商学院时用真实案例，研究、阅读、分析、小组讨论、代入到案例中的人在相同情况下，我会怎么样。
作者找了个大宗商品的暑期工作。

政治与经济的因果关系初见端倪

尼克松当选总统，戴高乐把美元都兑换成黄金，通过新闻和市场走向，作者开始理解政治与经济的因果关系。
1970年注意到黄金升值，汇率出现在新闻里。美元和黄金挂钩，那个时候作者对政府人员还是相信的。美元很文件，黄金是过时金属，投机者会引火自焚。
1971年8月15日，尼克松宣布美元和黄金脱钩，美元贬值。尼克松看起来采取行动在支撑美元汇率，但行动完全相反。
作者得出结论，政府越是坚决地做出承诺，局势也许越严重。

印钞与股市的关系

第二天股市上涨4%，和作者预期完全不一样。这个时候作者花了一个夏天，理解美元贬值和股市大涨的因果关系和必然。
作者理解到了以前发生过多次并可验证，作者的教训是：弄清楚其他时间、其他地点、其他人身上发生过的事情，要弄明白，这样才不会不知道如何应对。
1972年，大量印钞经济股市繁荣，漂亮50股收益很好。

美元贬值和股市涨价的因果关系

在几个月后经济放缓、油价暴涨、大宗商品价格暴涨。故事也在1973年下跌。
这个逻辑是60年代靠借债维持的开支持续到70年代，美元黄金脱钩，美国用贬值的纸币偿还债务，实际上是事实违约。
这位宽松信贷提供条件导致更多开支，金本位崩溃后通胀上升、大宗商品价格更高，1973年美联储收紧了货币政策，导致大萧条，严重的股价下跌和经济恶化

我从这件事得到的教训是：

所有人想法一样，就会反应在价格中，有可能犯错。所有行为一样会引起一个大体相同的反向反应和市场的反转。
类似的事情不断发生，事情总是情景在线，所有事情以符合逻辑的因果关系不断发生。
1973年股票不火，大宗商品火了，作者成为百年中型证券公司多米尼克的大宗业务主管，行情不佳很快倒闭
美国经济陷入困境，水门事件发生，作者再次看到政治和经济如何交织一起：经济先起作用，恶性循环导致悲观，卖掉股票继续下跌。
这个是1966年触顶的镜像。人们的共识是错的，实际上这个时候价格很低，政府采取行动改善，放松货币政策，在1974年触底。

《原则》作者经历(一)

发表于 2022-07-18 更新于 2022-09-29

导读

作者说他的成功在于无知的时候知道自己怎么做。基于某些原则，让他获得了成功。

原则的重要性？

  1. 面对需要快速做出反应的事件，拥有原则可以快速做出正确的反应，而不是每次都像第一次遇到一样。
  2. 依据清晰的原则做事，将我们自己认为合适的原则写下来，促使自己清楚知道自己原则，改进原则，做更好的决定，让别人更好理解我。

什么叫做拥有自己的原则？

独立思考并决定 想要什么、事实是什么、这个事实下如何实现自己愿望。
保持谦虚和心胸开阔，动用自己最佳思维。
原则需要真实反映你的性格和价值观，做到知行合一并不断反思。

作者的原则，怎么得到这些原则的？

成功的关键在于，知道如何努力追求很多东西，也知道如何正确地失败（经历失败后吸取教训）
作者不断学习和不断改进。

1. 用可信度加权的方式做决定

我怎么知道我是对的

2. 按照原则做事

清晰、明确、轻松地评估逻辑，可以判断自己是否知行合一

3. 用系统化的方式做决策

算法的形式将决策标准列出来，不带感情、更快，在可信度加权的情况下极其强大。

头脑中的决策

基于系统化做出的决策，让作者冲普通中产变成常规标准中的成功人士
给予了有意义的工作和人际关系，作者、作者原则、桥水独特运营方式得到大家的认可。

这本书分成三部分

第一部分：

作者的经历，读者请按照原则去映射他的故事，看到原则的逻辑和价值，并判断多大程度上适用我自己

第二部分

生活原则：讲述原则怎么应用于自然世界、私人生活、人际关系、企业经营和决策。
个人目标做出有效决策的五步流程
心理学和神经学理论，说明如何将这些理论应用到生活中

第三部分

桥水的运营方式
创业择优：极度求真、透明实现有意义的工作和人际关系
在微观层面如何实践。
基于工作原则，构建明智高效的决策

《微习惯——简单到不可能失败的自我管理法则》

发表于 2022-05-29 更新于 2022-05-30

微习惯

看上去是一个自我管理系统，从小习惯开始

目录：

  1. 微习惯是什么：习惯和大脑；培养好习惯
  2. 大脑的工作原理： 稳定大脑；聪明管理者；自动行为放逾期；模式探测器
  3. 动力VS意志力： 意志力打败动力原因；意志力工作原理
  4. 微习惯策略：运用意志力；拓宽舒适区；出现阻力时间点；跨越障碍
  5. 微习惯独特之处：现有习惯；微习惯+意志力；没有截止时间；提升自我效能感；自主感
  6. 彻底改变的八步骤：选择微习惯和计划；挖掘内在价值；纳入日程；回报机制；跟踪；微量开始超额完成；服从计划；留意习惯标记
  7. 微习惯八大规则：不自欺；满意每个进步；回报自己；保持头脑清醒；
    感觉强烈抵触后退缩小目标；提醒自己很轻松；不要小看；多余精力超额完成任务

大脑组成和微习惯原理

大脑由执行决策和自我行为模式识别两部分组成。

  1. 前额皮层管理功能相当活跃，但是需要消耗大量精力

  2. 基底神经节的自动功能，效率高，无需持续监督能处理各种任务。

微习惯策略的重点

远离消极的东西；靠近积极的东西

给生活注入好习惯，被动坏习惯通常会被替换。

微习惯，小的不可思议的一小步，可以触发正反馈，提升自我效能感，减轻抵触情绪，获得惯性。

压力大会促使你使用习惯做法，如果习惯很好，对自己来说是非常好的。

大脑分工和微习惯关系

大脑前额皮层容易疲劳，功能强大，消耗太多精力，当疲劳时，掌管重复的部分会接管大脑。这块是基底神经节，一旦我们训练基底神经节完成积极的重复行为，我们就能获得巨大的收益。

微习惯和意志力

越缺乏动力，你消耗的意志力越高；意志力消耗太高，你就难以持续。

  1. 动力并不可靠： 精力不足；很难培养；你不想；热情递减。所以不要依赖动力，有很好，没有也没关系。
  2. 意志力很可靠：能自我强迫；可以强化；可以通过计划执行。

意志力损耗的5个因素：努力程度、感知难度、消极情绪、主观疲劳、血糖水平。

微习惯几乎不会消耗意志力，能保证你的意志无穷无尽，同时在任何情况下都能启动一个微习惯。

  1. 努力程度： 自我损耗极小
  2. 感知难度： 没有难度
  3. 消极情绪：你几乎感觉不到因为要做微习惯导致的消极情绪
  4. 主观疲劳： 微量疲劳，一想到微目标，就感觉精力充沛
  5. 血糖水平：在低血糖和疲劳是，最能让你动起来的是微习惯

微习惯和阻力

  1. 阻力： 启动阻力（初始速度） 继续行动阻力（加速度）
  2. 自主权是微习惯的另一个重要的核心。
  3. 微习惯带来自我能效感、自主权、远离恐惧胆怯和忧郁、增强正念

微习惯的八步走

  1. 选择三个微习惯，评估并做好长期计划
    把习惯变成小的不可思议，小到让你感到荒唐。
  2. 给习惯赋予意义
  3. 设定计划
  4. 确定回报
  5. 记录
  6. 微量开始，超额完成
  7. 服从安排，摆脱高期待
  8. 留意习惯养成的标记： 不抵触、认同自己、行动不需考虑、不担心、常态化
    即使没有超额完成，你也是了不起的，超额完成了，你是很了不起。

微习惯设定的重要原则

  1. 不自欺，不要偷偷提高标准
  2. 满意每一步，大脑错觉是微习惯的重要核心，是一种相信小积累能量变到质变
  3. 经常回报自己，恢复意志力
  4. 保持清醒，完成了任务兴奋，执行时单调和重复，都应该平常心
  5. 提醒自己很轻松
  6. 相信微步骤
  7. 不要制订更大的目标，而应该专注任务

《学习之道》(十)总结：十个好的学习法则、十个坏的学习法则

发表于 2022-05-17 更新于 2022-05-30

释放无尽潜力

10个好的学习法则

  1. 运用回想 （小学语文总结段落思想、中心思想）
  2. 自我测试
  3. 对问题组块
  4. 间隔重复动作
  5. 练习中交替不同的解法
  6. 注意休息
  7. 使用解释性的提问和简单的类比（费曼提出）
  8. 专注
  9. 困难的事情最先做
  10. 心理对照

10个坏的学习法则

  1. 被动重复阅读
  2. 满眼重点标记
  3. 瞟一眼解题思路，就胸有成竹
  4. 火烧眉毛才学习
  5. 清楚解法，仍反复练习
  6. 与朋友把学习变成闲聊
  7. 做题忽视课本
  8. 有疑问不找导师和同学核对并解决
  9. 时常分心，还以为自己学的深入
  10. 睡眠不足

小结

分类、感想

如果对书分类，这是一本工具书，又不局限于工具书，

有句话是「假传万卷书真传一句话」。

就是说如果仅仅是工具书，告诉你怎么做就好了。

但是我们都是凡人，没有那么高的悟性。作者要让你心服口服，就把原理和推导过程告诉你，让你自己得出结论说服自己，方能让你心服口服。

内容评论

这本书引入大量的论证和参考文献去解释每一条法则，最终就是这一页的内容。如果没有经过前面的学习，我会有很多疑问和踌躇：要么动作不标准；要么半信半疑地做。

价值

目前我吸取到对我最有帮助的习惯有：学习要专注理解、每小节总结、搭建组块、间隔地重复、简化组块模型。

反思我学习习惯，搭建组块和重复的事情很少做，所以学得多，记住的很少，效果很差。

在之后的学习中，要好好的用到这些原理，并经常翻翻这本书，内化这本书，养成良好的学习习惯。

认识自己大脑的运行方式，充分发挥大脑潜力。

《学习之道》(九)如何准备考试

发表于 2022-05-17 更新于 2022-05-30

备考检查清单(经常做才能回答是)

  1. 尽力理解课本内容（带目的找例题不算）。
  2. 和同学一起解决作业问题，核对答案。
  3. 和同学合作之前，尝试自己大致写出每道题的解法。

备考

回答越多是，说明备考越好，两个和两个以上否要改变思路

  1. 积极参与家庭作业小组讨论（贡献点子，提出问题）。
  2. 遇到困难请求老师和助教。
  3. 交作业时完全理解所有问题的答案。
  4. 不明白解法时，在课上提问需求解释。
  5. 学习指导手册，考前有仔细通读过一遍吗，确保手册上所有问题都会做了吗。
  6. 尝试快速列出解题纲要，而不在基础计算上花时间。
  7. 一起复习学习指导手册和问题，并相互提问。
  8. 考前有复习题，是否参加过，在课上提出任何你不太确定的问题。
  9. 考前有合理的睡眠。

更多技巧

  1. 有难入简，先做男难的，能做多少是多少；然后做简单的；然后再做另一道困难的。 保证发散模式能充分发挥作用。
  2. 考试前克服恐惧：把计划B写出来，写下临考前自己的感觉和想法。
  3. 考试中克服焦虑：把注意力转移到呼吸上，手放到腹部，体会手的起伏。
  4. 考前浏览复习材料，重温一遍。
  5. 考中从后往前检查，正面的念头。
  6. 做题时停顿几秒，如闭眼5秒钟。

《学习之道》(八)塑造你的大脑、学习的复兴

发表于 2022-05-17 更新于 2022-05-30

形成直观的认知

  1. 在熟练运用组块资料之后，尽量不要纠结于每一个小细节，而是自然而然地去解决问题。
  2. 有些同学能快速掌握材料，给别人打击，但是平均水平的学生在主动性上、做好事情能力上和创造力占优。
  3. 把握创造力的关键是专注模式切换到更放松、白日梦一样的发散模式。

事实

  1. 大脑发育速度不一样，有些人25岁才完成发育。
  2. 在科学界很多重量人物，起初都是问题少年。
  3. 在科学数学技术领域取得成功的专业人士，逐渐习得一个特质，就是如何组块——提炼关键思想。
  4. 比喻和实体类比也能构建组块，组块可使截然不同的领域概念对另一个领域产生影响。
  5. 无论做什么职业，都要有开放的心态，保证自己学习宝库常备数学和科学知识，存储更多组块。

抽象和简化概念

  1. 心灵之眼非常重要，帮我们在大脑中排演，将学过的知识拟人化。
  2. 迁移是把所学知识从一个知识背景运用到别处的能力。
  3. 关键是要掌握一个数学概念的精髓，有利于概念迁移和将之应用到新途径中。
  4. 学习过程中一心多用让学习无法深入，限制迁移所学知识的能力。

学习的复兴

  1. 自主学习是一种最深入、最有效的学习方式
  2. 提高独立思考能力
  3. 在学习中毅力往往比智力更加重要
  4. 锻炼自己去解除你敬仰的人，你会结识知识渊博的导师，一席话可能改变你的未来。
  5. 如果你没有快速掌握学习材料的重点，不要灰心，学的慢的同学会抓住根本问题，真正理解前因后果能更深层次地理解问题。
  6. 人人是竞争和合作的关系，总有人批评和低估你付出的努力，请淡然处之。

团队合作

  1. 即使自信一切没问题，专注模式可能在不经意犯下致命错误。
  2. 和愿意提出不同见解的小伙伴一起讨论，找出想法中的方法；随机应变应对压力；确保自己真的理解了你想让别人结识的知识，巩固已有知识；建立重要的职业关系
  3. 客观对待学习中的批评，不要自欺欺人

物理学教授布拉德*罗斯对学习的建议

  1. 检查计算过程花的时间比解决过程花的时间少很多，多检查。
  2. 量纲分析是你的朋友，错误极有可能发生在计量单位不匹配的关键点上。
  3. 思考方程的意义，对计算结果有预期。
  4. 复杂因子，变量分别取极限，看是否对你的理解方程有意义。

想起高中时候考试，我都是先把所有题目都看一遍，尤其是大题目，尝试写下是哪个模块下的什么思路。然后才从头做简单的题目。

口述记忆宫殿法

  1. 挑选你熟悉的空间，如办公室布局，或者家庭布局，把它当成你的记事本
  2. 将要记忆的内容按顺序分别放到布局中的某个地方，并想象成一个鲜活的场景。
  3. 按照在空间布局中行进路径，多走几遍。

技巧

  1. 打造生动形象的比喻
  2. 间隔重复有助将概念向记忆存储
  3. 创建意群（如藏头诗一样，把整个内容抽成一句话）
  4. 别把窍门和真正知识混为一谈
  5. 讲故事
  6. 动手写写加强记忆
  7. 自言自语说给自己听
  8. 体育锻炼对神经元生长有很大促进作用
  9. 入睡前，回顾当天所学。

小技巧

  1. 整理环境、冥想避免干扰
  2. 改造为负面思维为正面思维
  3. 有负面小情绪可理解，立刻开始做
  4. 不开电脑、先看简单的题、抄下要解决的题随身带着直到脱不下去了。

行动日记

  1. 简单列出下个星期要做的关键任务清单，一周一次。
  2. 在第一天写出第二天的日行计划清单
  3. 每天都专注今天的任务。
  4. 先从最困难的事做起

玛丽查的数学浸泡实践

  1. 学习材料的某小节，所有题目做一遍，每种类型都练到
  2. 浸泡难题
  3. 用表格列出所有重要概念，给各类问题配上例题
  4. 考前列一张内容全面的表格，包含科目、小结题型、解题技巧

克服拖延

  1. 记录行程日记
  2. 每天承诺完成一定任务
  3. 睡前写下计划的任务
  4. 将工作细化成一系列笑挑战
  5. 相信自己的新时间系统

《学习之道》(四)组块的构建

发表于 2022-05-17 更新于 2022-05-30

解题有两种模式

按顺序推理和跳跃式的，而直觉是跳跃式的，大部分难题是由直觉解决的，需要用专注模式对直接进行验证，直觉并不总是正确的。

构建组块的重要性

  1. 一个过程或者思路理解透彻后，在宏观视角可视为一个整体，高度压缩后存进大脑，浓缩的思维让人获取洞察力
  2. 重复和练习的背后是固化组块，难点是枯燥
  3. 如果想对学习材料了如指掌，那么从上至下的宏观视角和从下至上的组块能力都不可或缺。

我们都喜欢创新，以及能够纵览全局的学习。但在数学和科学的学习中，必须进行适量的练习和重复，否则就无法构建组块来支撑专业技能科学的学习中，必须进行适量的练习和重复，否则就无法构建组块来支撑专业技能

提取知识

提取知识和回想知识帮助我们增加了学习深度，帮组我们实现组块。

如果一开始就画概念图之间关系，就是基础组块还没建立，就试着构建联结，完全是空中阁楼

活动一下，对难以理解的关键概念特别有用

整理组块

战胜自己， 整理课堂笔记和阅读笔记

与其过度学习，不如穿插学习

不要在同一个概念和技巧上投入太多时间学习，一旦理解了这个概念，就应该去学其他概念，隔几天在回顾这个概念。

如何搭建强有力的组块

  1. 全程纸上解决一个重难点问题 （要有线程答案，但是提交答案前不看）
  2. 重做一次，注意关键步骤
  3. 休息一下，启动发散模式
  4. 睡眠
  5. 再来一次，再次关注关键步骤（这个叫刻意练习）
  6. 给自己添加新题
  7. 主动重复，回想题目关键步骤

小技巧

  1. 越努力回想学习材料，你的记忆就越深刻；
  2. 提取练习是最强效的学习方式之一。
  3. 遭遇瓶颈，自然现象，越过后理解会更加深刻。
  4. 怎么备考，问题和答案梳理整齐，方便浏览；手写答案；主动重复。
  5. 测试是强效的学习：避免发蒙（工作记忆被填满），组块可以给工作记忆留出更多空间。

灵感项目经验

  1. 挖掘他人身上正能量特质
  2. 系统理解写下每个步骤
  3. 重复练习、发散模式专注模式切换
